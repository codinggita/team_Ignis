# 🔥 IGNIS JWAALA

> **Where Opinions Ignite. Where Voices Rule.**

IGNIS JWAALA is a **full-stack interactive web application** that combines **real-time polling, live discussions, and gamified faction systems** into a single immersive platform.

It transforms traditional voting systems into a **dynamic Vanguard Arena**, where users (called *Embers*) can vote, debate, and compete.

---

## 🚀 Project Overview

IGNIS JWAALA is designed to make opinion sharing:

* ⚡ Interactive
* 🔥 Competitive
* 💬 Engaging

Instead of simple polls, the platform introduces:

* Live voting systems
* Real-time discussions
* Faction-based identity
* Event-driven modules

---

## 🧩 Core Features

### 🗳️ Polling System

* Create and manage polls
* Multiple choice / single choice options
* Live voting with instant results
* Poll categories and duration control

---

### 💬 Live Chat & Debate

* Real-time discussion system
* Engage with other users (Embers)
* Debate topics dynamically

---

### ⚔️ Faction System

* Users belong to factions:

  * 🔥 IGNIS (Fire)
  * 🌊 AQUA (Water)
  * 🌱 TERRA (Earth)
  * 🌪 VENTUS (Air)
* Faction-based engagement and competition
* Leaderboards and dominance tracking

---

### 🔥 IGNIS PRESENTS (Events System)

* Daily active modules:

  * Live Poll Arena
  * Debate Room
  * Trending War
* Keeps platform dynamic and engaging

---

### 👤 Authentication System

* Login using:

  * GR Number
  * Date of Birth
* Protected routes and session handling

---
